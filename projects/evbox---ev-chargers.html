EVBox - EV Chargers Embedded Software

Project Details

Client

EVBox

Date

April 17, 2025

Budget

N/A

EVBox owned a portfolio of EV Charging solutions, spanning 3 technological generations and serving multiple geographical markets and use cases, requiring a high number of SKUs, thus complicating product support in software.

As Engineering Manager - Charge Foundations at EVBox, I was the line manager of 15 highly skilled and motivated engineers, belonging to 4 different teams, each covering specific areas of the product.

Foundation

  • Linux kernel
  • Image build system
  • Secure bootloader
  • Networking (WiFi, Ethernet, LTE-M)
  • Software update

GED

  • System state representation
  • Legal charge billing controller (Eichrecht)
  • User Experience (GUI, events, HMI, RFID, backlight, etc)

Powerboard

  • Power controller
  • Safety controller
  • Inter board communication protocol

Devops

  • Gitlab configuration
  • Runners configuration
  • Repo configuration
  • Repo templates (linters, static analysis, test scripts
  • HIL pipelines

‍

In 2021 I was one among the first to receive a Scaled Agile certification so I could help deploy the SAFe v5 Portfolio method of working.

I worked at EVBox, as a remote manager, from October 2020 to December 2024, at which point the AC business was wound down, assets divested, brand re-incorporated in a new entity which proceeded to become the new solely DC-focused EVBox headquartered in Bordeaux France.
